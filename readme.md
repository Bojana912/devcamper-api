# DevCamper API

> Backend API for DevCamper application, witch is directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update values/settings to your own

## Install Dependencies
```
npm install
```

## Run App
```
# Run in dev mode
npm run dev

# Run in production mode
npm start
```
